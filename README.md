# Decode Labs Prompt Engineering Internship

This repository contains projects completed during my Prompt Engineering Internship at Decode Labs.

## Project 1 – Data Extraction

Designed deterministic prompts to extract structured information from unstructured customer support tickets into JSON format.

**Skills Used**

* Zero-Shot Prompting
* Few-Shot Prompting
* JSON Schema Enforcement
* Data Extraction

## Project 2 – Chain-of-Thought Reasoning

Developed reasoning prompts that encourage structured thinking, verification, and self-correction.

**Skills Used**

* Chain-of-Thought Prompting
* Logical Reasoning
* Self-Verification
* Prompt Design

## Project 3 – Context-Anchored Answering (RAG Basics)

Built a closed-book question-answering workflow that answers only from provided documents and avoids hallucinations.

**Skills Used**

* Retrieval-Augmented Generation (RAG) Basics
* Context Grounding
* Hallucination Reduction
* Prompt Constraints
